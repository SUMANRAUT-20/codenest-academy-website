# codenest-academy-website
A responsive educational website built with HTML and CSS to practice semantic web design, reusable styling, and responsive layouts.

The website presents CodeNest Academy, a fictional IT school that introduces its courses, contact information, and location through a clean and easy-to-navigate interface.

## Project Purpose

The purpose of this project is to strengthen my understanding of:

- Semantic HTML structure
- Multi-page website development
- Reusable CSS styling
- Responsive web design
- Navigation between pages
- Form layout and user interface design
- Embedding external content such as Google Maps
- Managing source code with Git and GitHub

## Pages

- **Home** вЂ” Introduces CodeNest Academy and its featured courses
- **Menu** вЂ” Displays the available courses and learning options
- **Contact** вЂ” Provides an inquiry form for visitors
- **Access** вЂ” Shows the academy location using an embedded Google Map

## Main Features

- Responsive layout for desktop and mobile screens
- Consistent navigation and visual design across multiple pages
- Course information presented with images
- Contact form interface
- Embedded Google Map
- Separate CSS files for page-specific styling
- Shared styles through a common stylesheet

## How to Run the Project

No installation or package manager is required.

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in a web browser.

You can also use the **Live Server** extension in Visual Studio Code for automatic browser refresh while editing.
